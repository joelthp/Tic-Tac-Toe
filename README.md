# Jeu tic tac toe

## IMPORTANT ! 

Après avoir cloné/téléchargé le repo il faut 
- Faire un **bundle install**
- Lancer le morpion.rb 

### JEU

Tic Tac Toe fait pour 2 joueurs.
Il suffit de choisir la case (A1 ou C3 par exemple) chacun votre tour.
